1 Open your HTML file in a text editor

Right-click the .html file → Open with → Notepad (Windows) or TextEdit (Mac).
Select all with Ctrl+A (Windows) or Cmd+A (Mac), then copy.

2 Paste and generate a link

Go to the Publish tab. Type a name for the file, paste the HTML content, and click Generate Link.

3 Copy the link and add it to Moodle

Click Copy link. In Moodle: Add an activity → URL → paste the link → Save.
Students click the link and the page opens directly in their browser. No login needed.

Which method should I use?

Method 1 (Data URL) — the whole page is encoded inside the link. It is long but it works forever, even offline, with no external service. Recommended for all materials.

Method 2 (dpaste.org) — uploads to a free public service and gives you a short URL like dpaste.org/ABC123. Expires after 365 days. Good for sharing in chat or email where a long URL looks messy.

Can students see each other's content?

Method 1 links are completely unguessable — only people you share them with can open them. Method 2 links are technically public on dpaste.org but nobody will ever stumble on them (random strings). For exam materials or sensitive content, always use Method 1.

What file types work?

Only .html files. This includes everything produced in this course: the textbook chapter, the interactive 20-activity worksheet, the printable task sheet, and any future materials.
